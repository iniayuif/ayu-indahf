Algoritma Pencarian: BFS & DFS

April 2026 Selesai Python / JavaScript

Deskripsi Tugas

Tugas ini meminta mahasiswa untuk memahami dan mengimplementasikan dua algoritma pencarian dasar dalam kecerdasan buatan, yaitu Breadth-First Search (BFS) dan Depth-First Search (DFS). Kedua algoritma diuji pada sebuah graf tak berarah untuk menemukan jalur dari simpul awal ke simpul tujuan.

BFS (Breadth-First Search)

Menelusuri graf per level (lebar). Menggunakan struktur data Queue (FIFO). Menjamin jalur terpendek (jumlah edge) pada graf tak berbobot.

DFS (Depth-First Search)

Menelusuri graf sedalam mungkin sebelum backtrack. Menggunakan struktur data Stack (LIFO). Cocok untuk eksplorasi seluruh kemungkinan jalur.

Kesimpulan

BFS dan DFS adalah dua algoritma pencarian fundamental dalam Kecerdasan Buatan. BFS unggul saat kita membutuhkan jalur terpendek karena menelusuri level per level, sedangkan DFS lebih efisien dalam penggunaan memori dan cocok digunakan saat ruang pencarian sangat besar. Pemilihan algoritma tergantung pada karakteristik masalah yang dihadapi.
