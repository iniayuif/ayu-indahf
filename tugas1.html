Algoritma Pencarian: BFS & DFS

Penelusuran graf menggunakan Breadth-First Search dan Depth-First Search

BFS – Breadth First Search

Menelusuri simpul per level menggunakan Queue (FIFO). Menjamin jalur dengan jumlah edge terkecil pada graf tak berbobot.

DFS – Depth First Search

Menelusuri sedalam mungkin sebelum backtrack, menggunakan Stack (LIFO). Cocok untuk mengeksplorasi semua kemungkinan jalur.

Implementasi Python 

from collections import deque

def bfs(graph, start, goal):
    queue = deque([(start, [start])])
    visited = {start}
    while queue:
        node, path = queue.popleft()
        if node == goal:
            return path
        for nb in graph[node]:
            if nb not in visited:
                visited.add(nb)
                queue.append((nb, path + [nb]))
    return None

# Contoh:
graph = {'A':['B','C'],'B':['D','E'],'C':['F'],'D':[],'E':['F'],'F':[]}
print(bfs(graph, 'A', 'F'))  # ['A','C','F']
def dfs(graph, node, goal, visited=None, path=None):
    if visited is None: visited, path = set(), [node]
    visited.add(node)
    if node == goal: return path
    for nb in graph[node]:
        if nb not in visited:
            r = dfs(graph, nb, goal, visited, path+[nb])
            if r: return r
    return None

# Contoh:
graph = {'A':['B','C'],'B':['D','E'],'C':['F'],'D':[],'E':['F'],'F':[]}
print(dfs(graph, 'A', 'F'))  # ['A','B','E','F']

Perbandingan BFS vs DFS

Aspek BFS DFS
Struktur Data Queue (FIFO) Stack (LIFO)
Optimalitas ✅ Optimal ❌ Tidak optimal
Kompleksitas O(V + E) O(V + E)
Cocok untuk Jalur terpendek Eksplorasi penuh

Kesimpulan

BFS menjamin jalur terpendek menggunakan Queue, cocok ketika solusi optimal diperlukan. DFS menggunakan Stack dan lebih hemat memori, cocok untuk eksplorasi ruang besar. Pilihan algoritma tergantung kebutuhan masalah yang dihadapi.
