Minimax & Alpha-Beta Pruning

Algoritma keputusan optimal untuk permainan dua pemain (Tic-Tac-Toe)

Minimax

Memodelkan dua pemain: Maximizer (AI) dan Minimizer (lawan). Menelusuri seluruh pohon kemungkinan untuk menemukan langkah terbaik.

Alpha-Beta Pruning

Optimasi Minimax: memangkas cabang yang tidak relevan dengan batas alpha dan beta, tanpa mengubah hasil akhir.

Info: Setiap giliran AI, Minimax menelusuri semua kemungkinan papan. Alpha-Beta memangkas cabang bila α ≥ β, menghemat komputasi tanpa mengubah keputusan optimal.

Kode Alpha-Beta Pruning (Python) 

def minimax(board, depth, alpha, beta, is_max):
    score = evaluate(board)
    if score != 0 or not moves_left(board):
        return score

    if is_max:
        best = -1000
        for i, j in empty_cells(board):
            board[i][j] = 'O'
            best = max(best, minimax(board, depth+1, alpha, beta, False))
            board[i][j] = '_'
            alpha = max(alpha, best)
            if beta <= alpha: break  # ✂ pruning
        return best
    else:
        best = 1000
        for i, j in empty_cells(board):
            board[i][j] = 'X'
            best = min(best, minimax(board, depth+1, alpha, beta, True))
            board[i][j] = '_'
            beta = min(beta, best)
            if beta <= alpha: break  # ✂ pruning
        return best

Kesimpulan

Minimax menghasilkan keputusan optimal untuk permainan dua pemain. Alpha-Beta Pruning meningkatkan efisiensi dengan memangkas cabang tidak relevan, mencapai kompleksitas O(b^(d/2)) pada kasus terbaik dibanding O(b^d) Minimax murni.
